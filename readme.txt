== Description ==
You can set meta tags like keyword, description. You can customize on each individual page/post as well.

== Features ==
- FREE Plugin.
- One click to Install.
- Simple and Easy to Use
- Do not need to edit your theme's core files.
- Quick and supportive.
 
== Installation ==
1. Install as a regular WordPress plugin from plugin library or copy and paste this files within your wp-plugins folder.
2. Activate the plugin through the 'Plugins' menu in WordPress admin.
3. Check plugin is listed in dashboard menubar as "WP Simple Meta Tags".
